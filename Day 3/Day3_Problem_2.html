Var,Let and Const

1.var declarations are globally scoped or function scoped while let and const are block scoped.

var name = "Muthu"
{
let name = "Kumar"
console.log("Inside Block",name)
}
console.log("Outside Block",name)

Output:
Inside block : kumar
Outside block : Muthu

2.var and let variables can be updated.const variables can not be updated

var a = 10
let b = 5
const c = 1
a = 5
b = 10
console.log('a : ',a)
console.log('b : ',b)
c = 2

Output:
a : 5
b : 10
Uncaught TypeError: Assignment to constant variable.

3.While var and let can be declared without being initialized, const must be initialized during declaration.

var a
let b
a = 5
b = 10
const c

Output:
Uncaught SyntaxError: Missing initializer in const declaration
