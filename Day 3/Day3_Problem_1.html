Console Functions

1.Console.log("Hello")

Console.log is used to print the message in the console window.

2.Console.error("Error Occured")

Console.error is used to print the error message in the console window.

3.Console.warn("Warning message")

Console.warn is used to display the warnings in the console window.

4.Console.clear()

Console.clear is used to clear the console window.

5.Console.time() && console.timeEnd()

These functions are used to display the total running time

6.Console.table({'Name':'Muthu','Age':20})

This function is used to display the object data in the form of table.

7.console.group() && console.groupend()

These function are used to group the console functions.
