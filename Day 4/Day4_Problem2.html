Java Script

String Methods

		1.charAt() - is used to find the character at the given index.
		2.charCodeAt() - is used to find unicode of the character at the given index.
		3.concat() - this is used to join or concat to string.
		4.indexOf() - It is used to get a index value of a particular char.
		5.lastIndexOf - It provides the last occurance of the given letter.
		6.search() - used to search the input in the given string.
		7.match() - this is used to find the match in the given string.
		8.replace() - This replace function is used to replace the particular text in the string.
		9.substr() - It is used to fetch the part of the string from given string.
		10.slice() - is used to fetch the part of the given string.
		11.toLowercase() - used to convert a string to lower case.
		12.toUppercase() - used to convert a string to upper case.
		13.toString()  - it provides the string primitives the particular object.
		14.split() - used to split the string.
		15.trim() - used to remove the spaces in the string.

Array Methods

		1.push() - add new elements to an array at the end.
		2.pop() - returns the value that was popped out.
		3.shift() - removes the first element from array.
		4.unshift() - adds new element to an array at the beginning.
		5.length - used to identify the length of the array.
		6.delete - used to delete element from array.
		7.concat() - used to join to arrays.
		8.slice() - used  to slice particular piece of an array.
		9.reverse() - used to reverse the array.
